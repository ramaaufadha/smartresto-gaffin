# SmartResto SmartResto Gaffin Tech Demo

Modern single-restaurant digital ordering UI for UMKM, built with React + Vite + TailwindCSS.

## Tech Stack

- React
- Vite
- TailwindCSS

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open your browser at:

```text
http://localhost:5173
```

## Production Build

```bash
npm run build
npm run preview
```
